Сервис для мониторинга курса криптовалют с криптовалютной биржи.

Для запуска сервиса необходимо: 
Cначала скачать базу данных с https://github.com/BlinDogg/DBburseRateVer2 и добавить папку в свой "xampp" по пути:"C:\xampp\mysql\data\all_price".
Также нужно не забть запустить sql сервер
Далее запускаем через терминал файл wsForAll.py
И в завершении запускаем файл main.py


Get-запросы на сервер:
http://127.0.0.1:5000/   #Главная страница
http://127.0.0.1:5000/courses  #Все курсы из списка
http://127.0.0.1:5000/pare?coin={ADD1}&valute={ADD2}   #конкретный курс валюты, где ADD1 - криптомонета из списка, а ADD2 - Валюта из списка

Поддерживаемые пары: {'BTC': 'KZT'},{'BTC': 'GEL'},
                     {'ETH': 'RUB'},{'ETH': 'USD'}
                     {'USDT': 'EUR'},{'USDT': 'KZT'},{'USDT': 'GEL'},
                     {'USDT': 'RUB'},{'USDT': 'USD'}
